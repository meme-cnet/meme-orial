# Memeorial

Landing del proyecto (MEME)ORIAL — colección NFT en OpenSea.

## Arrancar en local

Necesitas Node 20+ y pnpm instalado.

```bash
pnpm install
pnpm dev
```

La app queda en `http://localhost:3000`.

## Producción

```bash
pnpm build
pnpm start
```

También puedes usar `node server.js` si despliegas con servidor custom.

## Estructura rápida

- `/` — home con carruseles y roadmap
- `/the-concept` — manifiesto del proyecto

Las imágenes de los carruseles están en `public/images/` y los listados en `src/data/`.
